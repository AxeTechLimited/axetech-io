# axetech-io

Official website for Axe Tech Limited. Built with Next.js, deployed to Vercel.

## Stack

- Next.js 14 (App Router) with TypeScript
- CSS variables from the Axe Tech Design System v3
- Deployed on Vercel, DNS at GoDaddy

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

Pushes to `main` deploy automatically via Vercel. Custom domain: `www.axetech.io`.
